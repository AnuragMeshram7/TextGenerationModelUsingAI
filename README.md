# AI Text Generation Chatbot

## Overview
An AI-powered chatbot developed using Python and Tkinter GUI. The chatbot integrates with Ollama API and LLaMA 3.2 model to generate human-like responses.

## Features
- Modern Dark Mode GUI
- Real-time AI Response Generation
- Ollama API Integration
- User-Friendly Interface
- Scrollable Chat History

## Technologies Used
- Python
- Tkinter
- Ollama API
- LLaMA 3.2
